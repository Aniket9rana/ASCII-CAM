# ASCII-CAM

ASCII-CAM is a web application that converts your webcam feed into ASCII art in real-time. It provides a variety of controls to customize the output, allowing you to create unique and artistic representations of the video stream.

## Features

*   **Real-time Conversion:** Converts your webcam feed to ASCII art instantly.
*   **Character Sets:** Choose from multiple character sets for different visual styles:
    *   ULTRA
    *   DETAIL
    *   STD
    *   MINIMAL
    *   BLOCKS
    *   BRAILLE
    *   MATRIX
    *   HEX
*   **Density & Scale:**
    *   **Columns:** Adjust the number of characters per line.
    *   **Zoom:** Zoom in and out of the video feed.
*   **Color Tone:** Apply different color themes to the ASCII art.
*   **Image Adjustments:** Fine-tune the image with controls for:
    *   Brightness
    *   Contrast
    *   Gamma
    *   Sharpen
*   **Effects:** Add creative effects to the output:
    *   EDGE
    *   SCANLINES
    *   GLITCH
    *   PIXELATE
    *   THERMAL
    *   GHOST
    *   RAIN
*   **Options:**
    *   **Mirror:** Flip the video horizontally.
    *   **Invert:** Invert the colors of the ASCII art.
    *   **Full Color:** Use RGB colors for each character.
    *   **Light Background:** Use a white background for better sharing.
    *   **Bold Chars:** Use a heavier font weight for the characters.
*   **Export & Actions:**
    *   **Freeze Frame:** Pause the video feed to capture a specific moment.
    *   **Save Screenshot:** Save the current ASCII art as a PNG image.
    *   **Save for Sharing:** Save the ASCII art with a light background, ideal for sharing.
    *   **Copy ASCII Text:** Copy the generated ASCII characters to the clipboard.

## How to Use

1.  Open the `index.html` file in a web browser.
2.  Allow the browser to access your webcam.
3.  Use the controls in the right-hand sidebar to customize the ASCII art.
4.  Use the action buttons to freeze, save, or copy the output.
